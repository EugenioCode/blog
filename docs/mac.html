    Mac软件推荐

    jianjun.yang 4/5/2022 Mac

    # 自用Mac软件整理

    # 图像处理

    # 截图

    • iShot (opens new window) []

      目前一直在用的截图软件，支持滚动截图、录屏、图钉等

    • 截图 (opens new window) []

      腾讯出的截图软件，就叫「截图」，支持录屏、导出gif功能

    • Xnip (opens new window) []

      软件支持滚动截图默认有水印，需要购买或订阅 Pro 版才能去除水印

    # 画图

    • excalidraw (opens new window) []

      开源软件，人气很高，一直在维护 操作简洁，有很多素材库可直接使用 很棒的手画风格，看着非常舒服

    • ProcessOne (opens new window) []

      免费在线流程图思维导图 可用于原型图、UML、BPMN、网络拓扑图等多种图形绘制

    # 图床

    • PicGo (opens new window) []

      目前一直在用的图床工具，图片存储在github仓库，上传成功后会自动生成markdown语法的图片链接，直接粘贴行，很方便。

    # 前端开发软件

    # 编辑器

    • VSCode (opens new window) []

    • Webstorm (opens new window) []

    # 接口调试

    • Postman (opens new window) []

    # 数据库

    • Navicat Premium

    • Robo 3T

    # 工具

    # 终端

    • iTerm2

    • OpenInTerminal (opens new window) []

      开源软件

      能够使用VSCode快速打开项目

      将终端快速定位到当前目录

    • Fig (opens new window) []

      开源软件

      终端与VSCode命令行的提示

    # 菜单栏

    • Hidden Bar (opens new window) []

      一个超轻的 MacOS 实用程序，有助于隐藏菜单栏图标,当顶部菜单很多的时候，这个软件就很实用了，可以在应用商店免费下载

    • uBar (opens new window) []

      类似windows的底部菜单栏

    # 解压缩

    • Keka (opens new window) []

      在App Store里面下载需要30元，在官网免费下载，丑但是很好用的解压缩神器

    # 视频播放器

    • IINA (opens new window) []

      苹果 Mac 平台上最强的好用万能视频播放器

    # 系统清理

    • Clean My Mac X (opens new window) []

    • Tencent Lemon (opens new window) []

    # 分屏

    • Magnet (opens new window) []

      Magnet 一款Mac下的窗口分屏软件，让当前活动窗口快速变换位置到屏幕左边、屏幕右边或者角落，中间，全屏宽等功能

    # 剪切板

    • Paste (opens new window) []

      Paste for Mac是Mac平台上一款专业的剪切板记录增强工具,它能够为您储存您在设备上复制您的所有内容,并将其储存在Paste Mac的历史记录中
